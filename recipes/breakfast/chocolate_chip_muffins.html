Chocolate Chip Muffins

Ingredients X1 X3 X4
Flour 1-3/4 C 5-1/4 C 7 C
Baking Powder 1 T 3 T 4 T
Salt ½ t 1-1/2 t 2 t
Margarine ¼ C ¾ C 1 C
Sugar ½ C 1-1/2 C 2 C
Eggs 1 3 4
Milk ¾ C 2-1/4 C 3 C
Vanilla 1 t 3 t 4 t
Chocolate Chips 1 C 2 C 3 C

Instructions:

  1. Cream margarine, sugar, and eggs together.
  2. Add milk and dry ingredients.
  3. Stir in chocolate chips.
  4. Bake at 400 degrees for 20 minutes.
